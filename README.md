## ⚡ Getting Started

### Prerequisites

- Node.js **18+**
- npm or yarn

### 1. Clone the Repository

```bash
git clone https://github.com/shreyashgautam/Portfolio_shreyash.git
cd Portfolio_shreyash
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Locally

```bash
npm run dev
```

Open **`http://localhost:5173`** in your browser.

### 4. Build for Production

```bash
npm run build
```
